# 요구사항

## 초기 요구사항
- 아이폰 Safari 한글 IME 문제 해결을 위한 웹 터미널
- 별도 입력 버퍼(input field)에서 한글 조합 후 터미널에 전송
- 모바일 키보드 열어도 화면 스크롤 방지 (마지막 명령이 화면 맨 밑에)
- 세션 유지 (브라우저 재접속 시 복원)
- 프롬프트 `$ ` 로 간결하게

## UI 요구사항
- Send 버튼 + Enter 버튼 분리
- Send/키보드리턴 시 텍스트 전송 후 `\r` 별도 전송 (50ms 딜레이, CLI 호환성)
- 빈 입력 필드에서 Enter → `\r`만 전송
- 특수키 2열 구조: Quick keys(항상) + 확장 패널(토글)
- Quick keys: 탭 드롭다운, 1, 2, 3, ESC, TAB, ^C, Keys, ⇣(맨아래스크롤, 파란색)
- ^C 표기 (C-c 아님)
- ^D, ^Z, |, /, -, ~ 는 quick keys에서 제거

## 멀티탭
- 드롭다운으로 탭 관리 (추가/전환/닫기)
- 각 탭 독립 pty + scrollback 버퍼

## 인증
- `--auth` 플래그로 선택적 활성화
- ID/Password, JWT, SQLite(bcrypt)
- JWT_SECRET 환경변수에서 읽기 (없으면 오류 표시)
- Remember me: 브라우저 자동완성 (localStorage 아님)
- "USE AT YOUR OWN RISK" 경고바 + 숨기기 버튼 + 로그아웃 버튼
- `-w, --cwd` 옵션으로 시작 디렉토리 설정

## 스크롤
- 출력 중 스크롤업 시 자동 맨아래 방지 (사용자 위치 유지)
- 터치 모멘텀 스크롤 (관성)
- smoothScrollDuration: 80, scrollSensitivity: 2
- 맨아래스크롤 버튼 (⇣, 파란색, Keys 옆)

## 네트워크
- JWT 토큰으로 자동 로그인 (네트워크 끊김 복원 시에도)
- WebSocket 자동 재접속 (끊김 시 1초~10초 간격)
- 토큰 만료 시에만 삭제 (네트워크 끊김으로 삭제하지 않음)
- `/api/verify` 엔드포인트로 토큰 검증

## 서버
- 정적 파일 캐시 비활성화 (Cache-Control: no-store)
- null 바이트 메시지 필터링

## SSL
- `--ssl` 옵션으로 HTTPS 활성화
- `--ssl-cert`, `--ssl-key`로 커스텀 인증서 경로 지정 (기본: `certs/`)
- 자체서명 인증서 생성: `openssl req -x509 -newkey rsa:2048 -nodes -keyout certs/key.pem -out certs/cert.pem -days 365 -subj "/CN=<host>" -addext "subjectAltName=IP:<host>"`

## 모바일 안정성 요구사항 (iPhone Safari)

### 키보드/뷰포트
- [ ] **키보드 열림 시 스크롤 점프 방지** — 입력 필드 포커스 → 키보드 열림 → visualViewport 리사이즈 → fitAddon.fit() → baseY 변경 → 스크롤이 중간으로 튀는 문제. scrollToBottom 타이밍이 xterm 내부 리사이즈보다 빨라서 발생
- [ ] **키보드 닫힘 시 레이아웃 복원** — 키보드가 닫히면서 뷰포트가 다시 커질 때 터미널 영역이 정상 복원되는지
- [ ] **iOS 주소창 표시/숨김** — Safari 스크롤 시 주소창이 나타나거나 숨겨지면 실제 뷰포트 높이가 변하고 레이아웃이 깨질 수 있음
- [ ] **화면 회전(가로/세로)** — orientation 변경 시 터미널 크기 재계산 및 스크롤 위치 유지

### 터치/제스처
- [ ] **pull-to-refresh 방지** — 터미널 영역에서 아래로 당길 때 Safari의 새로고침이 트리거되면 안 됨
- [ ] **rubber-band 스크롤 간섭** — iOS의 바운스 스크롤이 터미널 스크롤과 충돌
- [ ] **더블탭 줌 방지** — 버튼 빠르게 두 번 탭 시 확대되면 안 됨
- [ ] **롱프레스 컨텍스트 메뉴** — 터미널 영역 길게 누를 때 iOS 텍스트 선택/복사 메뉴가 뜨는 것 제어
- [ ] **터미널 스크롤 vs 페이지 스크롤 충돌** — 터미널 맨 위/아래 도달 시 페이지 자체가 스크롤되면 안 됨

### 연결/백그라운드
- [ ] **앱 백그라운드 전환 시 WebSocket 끊김** — iOS가 백그라운드 탭의 WebSocket을 종료함. 포그라운드 복귀 시 자동 재접속
- [ ] **화면 잠금/해제 후 복원** — 잠금 해제 후 터미널이 정상 동작하는지 (WebSocket 재접속 + 스크롤백 복원)
- [ ] **저메모리 상황** — iOS가 메모리 부족 시 백그라운드 탭을 kill. 페이지 재로드 시 토큰으로 자동 로그인 필요

### 입력
- [ ] **한글 조합 중 전송 방지** — isComposing 체크가 모든 시나리오에서 동작하는지 (Safari IME 특이 동작 포함)
- [ ] **클립보드 붙여넣기** — 모바일에서 긴 텍스트 붙여넣기 시 정상 동작
- [ ] **입력 필드 포커스/블러 반복** — 빠르게 포커스/블러를 반복할 때 레이아웃이 흔들리지 않도록

### 디스플레이
- [ ] **Safe Area (노치/Dynamic Island)** — 노치/홈 인디케이터 영역과 UI 겹침 방지
- [ ] **다크모드/라이트모드 전환** — iOS 시스템 테마 변경 시 깨지지 않는지
